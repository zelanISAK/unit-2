# Quiz 24
Create a program shows the graph of the parabola for 100 values of x in the interval -10 < x < 10.

$f(x) = 2(x+5)^{2}$

## Code
```
from matplotlib import pyplot as plt
import warnings
warnings.simplefilter('ignore')

x,y,s=[],[],-10.2
for i in range(100):
    s += 0.2
    x.append(s)
    temp = (s+5)**2
    y.append(temp)

plt.ylabel('$f(x) = 2(x+5)^2$')
plt.xlabel('x')
plt.plot(x,y)
plt.show()
```
