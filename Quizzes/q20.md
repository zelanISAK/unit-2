# A. Create a function that produces the table of truth or 3 inputs:

### Code
```
def get_truth():
    i=0
    output = "| A | B | C |"
    A,B,C=False,False,False
    temp=1
    for i in range(8):
        output += '\n| ' + str(int(A)) + ' | ' + str(int(B)) + ' | ' + str(int(C)) + ' |'
        if temp>0 and temp%4==0:
            A=not A
        if temp>0 and temp%2==0:
            B=not B
        C = not C
        temp+=1
    return output
```
